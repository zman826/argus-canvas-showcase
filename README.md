# Argus Canvas Showcase

シーマ社の事例検索サイネージ。音声駆動でシーマ導入事例ページを即時表示する
Windowsアプリ。Gemma 4をローカル動作させ、完全オフラインで稼働します。

## 開発元
株式会社アーガス企画

## 技術スタック
- WPF / .NET 8
- Ollama + Gemma 4 E4B
- ChromaDB
- Python 3.11+
